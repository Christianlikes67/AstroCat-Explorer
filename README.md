# AstroCat Explorer

## About

AstroCat Explorer is a simple web application that uses NASA's Astronomy Picture of the Day (APOD) API. The project lets users view NASA's daily image, search for images from other dates, and explore space.

I made this project to practice working with APIs, JavaScript, and CSS animations while creating something more interesting than a basic image gallery.

## Features

* View NASA's Astronomy Picture of the Day
* Search for images by date
* Get a random space mission
* Earn XP while exploring
* Responsive design

## Built With

* HTML
* CSS
* JavaScript
* NASA APOD API

You can get a free API key here:

https://api.nasa.gov/
