# AstroCat-Explorer
🐱🚀 Cosmic Cat Command

Cosmic Cat Command is a fun and interactive web app that lets you explore NASA's Astronomy Picture of the Day (APOD) with a unique space-cat theme.

Instead of creating a standard NASA gallery, this project combines space exploration, animated visuals, and a playful personality to create a more engaging experience.

✨ Features
🌌 NASA Astronomy Picture of the Day API
📅 Search for a specific date
🎲 Random Mission button
🐱 Random Space Cat facts
🏆 Mission counter
⭐ XP system with Space Cat ranks
☄️ Animated shooting stars
🪐 Floating planets
🚀 Rocket cursor
💎 Glassmorphism UI
📱 Responsive design
🎨 Custom space-themed interface
⚡ Smooth animations and transitions
🛠️ Built With
HTML5
CSS3
JavaScript (ES6)
NASA APOD API
